# Website Design Guidelines

The following are the design guidelines for this project, which apply to it and therefore should guide every single decision and change.

## Aesthetic

It is simple, clean, and consistent. It aims for a "nerdy and professional" vibe with a single, clean and bold monospace typeface (like JetBrains Mono), and a strict black-and-white color palette with a single accent color (TC39 Orange: #FC7C00). The styling relies heavily on sharp, brutalist geometry rather than soft shadows or rounded corners. Borders are thick, well-defined (e.g. 2-4px solid), and shapes are angular.

## Emoji Constraints

The site uses country flags and icons throughout, but Windows lacks native support for emoji flags. We must include a fallback (like Twemoji) to reliably render flag emojis cross-platform.

## Responsive Web

The website should work well on all popular screen sizes and we should test to ensure that the design doesn't break on desktop or mobile at any time.

## Dark Mode

The website has a dark and light mode, it should work well in both and every feature should keep this in mind.
